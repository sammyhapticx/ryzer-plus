# Ryzer Plus
An image resizing tool that aims to keep every bit of quality intact.
It is a fast, no-quality-loss image resizer built with vanilla HTML, CSS, and JavaScript. Upload images, resize with aspect ratio lock, preview side-by-side, and download crisp PNGs—perfect for web devs, gamers, and quick edits.

**[Play live on GitHub Pages](https://sammyhapticx.github.io/ryzer-plus/)**

## 🚀 Features
- **Drag & drop** or file upload (JPG, PNG, GIF, WebP).
- **Max width/height sliders** with optional aspect ratio lock.
- **Side-by-side preview** (original vs resized).
- **High-quality canvas rendering** (`imageSmoothingQuality = 'high'` for sharp results).
- **PNG download** (lossless) or tweak for JPEG compression.
- **Mobile-responsive** dark theme; works offline after load.
- Single `index.html` file—no build tools needed.
